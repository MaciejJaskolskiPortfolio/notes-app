# Notesapp

Simple application for creating daily notes.

User can create, edit or delete a note with following properties: title, description, date, icon and color.
User can view notes by selecting a date.

App was tested on Mozilla Firefox and Chrome.

Programmed with Angular 8 and SCSS.
Tested with Karma/Jasmine.

CI/CD done with TravisCI and Heroku.
